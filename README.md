# Netflix_Dashboard_Tableau
Tableau Dashboard on Netflix
	About: Designed and developed an interactive Netflix dashboard using Tableau to visualize and analyze global entertainment data across multiple KPIs, including country-wise TV shows & movies, top 10 content top ratings, description(rating, release year, date added, duration, content type and description of title) of every movie/ tv shows and movies and tv shows distribution year wise.
•	Tools: Tableau, Excel
•	Components: Symbol Map, Column Charts, Bar Charts, Slicer and Pie Charts with interactive filters and hyperlinks
